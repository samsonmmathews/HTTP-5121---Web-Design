Intro to flexbox - nesting layouts

Recall: Different properties are used for the flex container and for the flex items. You CANNOT set flex items properties on a flex container and vice versa. Since this is the case, if you need to nest, you need to nest as follows:

  • Outer flex container
    • Flex item
    • Flex item
      • Inner flex container
        • Inner flex item
        • Inner flex item
    • Flex item
